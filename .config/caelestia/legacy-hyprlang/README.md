# legacy-hyprlang

Old hyprlang user-override stubs (`hypr-user.conf`, `hypr-vars.conf`), both
empty. Replaced by `../hypr-user.lua` and `../hypr-vars.lua` after the Hyprland
Lua migration (commit `5b8f082`). Under the Lua config these `.conf` files are
no longer read. Kept only for reference / symmetry with `hypr/legacy-hyprlang`.
